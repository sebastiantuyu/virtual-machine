# Notes

### Instructions

> Instructions on /stack-vm.cpp are formated as:
```
	header: 2 bit 
	data: 30 bit;

# *******************
	header format:
		0 => positive integer
		1 => primitve instruction
		2 => negative integer
		3 => undefined
```

### Bitwise operations

>  A bitwise depends are operations at bit-level
>  is maded for creating operations only with bits and bytes

#### Main operators

- < <  Left shift 
- > >  Right shift 
- &		 AND operator
- +		 PLUS operator
- |    OR operator
